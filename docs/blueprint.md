# **App Name**: VisitaLog Pro

## Core Features:

- Welcome Dashboard: Displays the current date and time prominently, along with two distinct action buttons for 'Entrada Transportista' and 'Entrada Personal' to initiate the respective registration flows.
- Transportista Entry Flow (Simulated): A multi-step form simulation for registering transportistas (e.g., vehicle details, driver information, purpose of visit), utilizing mock data for demonstration purposes without real database interaction.
- Personal Entry Flow (Simulated): A streamlined form simulation for personal visitors (e.g., visitor name, host name, purpose of visit), showing mock data processing without actual storage.
- Simulated Visitor Log: A display page that lists recently 'registered' visitors (both transportistas and personal), pulling from local mock data and demonstrating how entries would appear.
- Dynamic Navigation System: Implement a clear and intuitive navigation structure enabling smooth transitions between the welcome dashboard, entry forms, and the simulated visitor log.
- AI Mock Data Generator Tool: A generative AI tool that can populate simulated forms and the visitor log with realistic-looking visitor names, company details, vehicle information, and visit purposes to aid in visualization and testing.

## Style Guidelines:

- Color Scheme: A professional and clean light theme for corporate use cases.
- Primary Color: A deep, authoritative blue (#255BA3), chosen for its association with trust and professionalism, ensuring strong contrast against lighter backgrounds.
- Background Color: A very light blue-grey (#F3F6F8), providing a subtle, unobtrusive base that maintains harmony with the primary color while ensuring excellent readability.
- Accent Color: A bright, clear cyan (#1DBCCD), acting as an invigorating counterpart to the primary blue, used for interactive elements and highlights to draw attention.
- Primary font: 'Inter' (sans-serif), for its modern, legible, and neutral aesthetic suitable for both headlines and body text in a business context.
- Use modern, minimalist vector icons that clearly represent their actions, focusing on simplicity and clarity consistent with a professional application.
- Employ a clean, grid-based layout for consistent spacing and easy readability, with prominent cards or sections for each key application area, optimizing for both desktop and mobile responsiveness.
- Implement subtle and quick animations for screen transitions and button hovers, enhancing user feedback without being distracting, aiming for a smooth and responsive feel.
